## 基于区块链的智能集装箱应用系统

### 目录
```

..src/
....robots.txt
....index.html * 入口
....app/ * 内容
......common/ * 公共组件
......components/ * 模块
......app.config.js * 配置
......app.constants.js * 常量
......app.module.js * module
......app.route.js * 路由
....assets/ * 资源
......locales/ * 国际化文件
......images/ * 图片

```


#### 安装
```
 npm install
 bower install
```

#### 运行
```
gulp run:dev
```
